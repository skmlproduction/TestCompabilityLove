# Индекс референс-скриншотов

Скриншоты **оригинального** приложения (Unity 3.0.5) для уточнения UX при F1/F2.  
**Не** использовать как ассеты в новом приложении (авторские права).

## Структура

| Папка | Содержимое |
|-------|------------|
| `play/` | Витрина Google Play (если сохранили с веба) |
| `device/` | Проход по всем тестам с устройства / эмулятора |

## Как заполнять

1. Сохраните PNG в соответствующую подпапку.
2. Добавьте строку в таблицу ниже.
3. Обновите `docs/product/screens_catalog.csv` и при расхождении — `SCREEN_INVENTORY_AND_NAVIGATION.md` (F1 уже создан, см. §6 в `REFERENCE_SOURCES.md`).

## Таблица

| файл | предполагаемый screen_id | заметки |
|------|--------------------------|---------|
| *(пусто)* | — | Добавьте первый файл, напр. `device/01_hub.png` → `hub_main` |
